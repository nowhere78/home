Open Agents는 Vercel에서 백그라운드 코딩 에이전트를 구축하고 실행하기 위한 오픈 소스 참조 애플리케이션입니다. 웹 UI, 에이전트 런타임, 샌드박스 오케스트레이션 및 GitHub 통합 기능을 포함하여 노트북을 사용하지 않고도 프롬프트에서 코드 변경까지 바로 진행할 수 있도록 지원합니다.
이 저장소는 블랙박스처럼 취급하기보다는 포크하고 수정할 수 있도록 만들어졌습니다.

Open Agents는 3계층 시스템입니다.
Web -> Agent workflow -> Sandbox VM
[111] (626,782)<clipboard-copy aria-label='Copy' class='ClipboardButton btn btn-invisible js-clipboard-copy m-2 p-0 d-flex flex-justify-center flex-items-center' role='button' />
웹 애플리케이션은 인증, 세션, 채팅 및 스트리밍 UI를 처리합니다.
해당 에이전트는 Vercel에서 내구성이 뛰어난 워크플로로 실행됩니다.
샌드박스는 실행 환경입니다. 여기에는 파일 시스템, 셸, Git, 개발 서버 및 미리 보기 포트가 포함됩니다.
