TABPFN_MODEL_CACHE_DIR
다운로드한 TabPFN 모델을 캐싱할 사용자 지정 디렉터리(기본값: 플랫폼별 사용자 캐시 디렉터리)
TABPFN_ALLOW_CPU_LARGE_DATASET
대규모 데이터셋(샘플 1000개 이상)을 사용할 때 TabPFN을 CPU에서 실행할 수 있도록 허용합니다.
true
CPU 제한을 무시하려면 이 값을 설정하십시오. 참고: 이 경우 속도가 매우 느려집니다!
PyTorch 설정:
PYTORCH_CUDA_ALLOC_CONF
: PyTorch CUDA 메모리 할당 설정을 통해 GPU 메모리 사용량을 최적화할 수 있습니다(기본값:
max_split_size_mb:512
).

		[14] (430,192)<font >PyTorch CUDA 설명서를 참조하십시오. />
