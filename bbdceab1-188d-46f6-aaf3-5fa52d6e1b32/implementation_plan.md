# 📖 Implementation Plan: Alpha Bible Portal (V5.6 Premium)

사용자님이 언제 어디서든 NIV 성경을 읽고, 마음에 드는 구절을 즉시 **'명품 쇼츠'**로 만들 수 있는 프리미엄 웹 포털을 구축합니다.

## 🎨 디자인 컨셉: 'Divine Minimalism'
- **배경**: 칠흑 같은 흑색(Obsidian Black)과 부드러운 유리 효과(Glassmorphism).
- **포인트**: 우아한 골드(Gold Leaf) 액센트와 세리프체(Playfair Display) 적용.
- **감성**: 성경의 권위와 현대적 세련미가 조화된 하이엔드 UI/UX.

---

## 🛠️ 주요 기능
1. **NIV 성경 열람**: 구약/신약 권별, 장별 선택 기능.
2. **반응형 리더**: 스마트폰과 PC 어디서든 최적화된 독서 경험.
3. **'쇼츠 제작' 원클릭 연동**: 읽고 있는 구절을 버튼 하나로 우리 AI 파이프라인(`luna_shorts_pipeline.py`)에 전송하여 영상 제작.

---

## 📅 개발 로드맵

### 1단계: 프론트엔드 구축
- [NEW] `src/web/bible_portal.html` (HTML + CSS + JS 통합 SPA)
- 프리미엄 다크 모드 테마 및 애니메이션 구현.

### 2단계: 성경 데이터 엔진 연동
- 공개 API (`bible-api.com`) 및 로컬 NIV 데이터셋 연동 테스트.
- 장/절 검색 및 자동 스크롤 기능.

### 3단계: 비즈니스 파이프라인 통합
- 선택한 구절을 젬마에게 전송하는 API 서버(간이) 구축.
- 텔레그램 알림 시스템과 연동.
