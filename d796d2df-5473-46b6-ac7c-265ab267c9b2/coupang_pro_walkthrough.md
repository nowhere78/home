# 🏆 Alpha Coupang Pro 워크스루

유튜브 영상 속 "고수"들이 사용하는 **트렌드 분석 → 상품 선정 → 자동 영상 제작** 방식을 파이썬 프로그램 하나로 완전히 통합했습니다. 사용자님의 요청대로 **"한 번 실행에 최소 3개의 영상을 연속으로 뽑아내는"** 기능까지 탑재 완료했습니다.

## 🚀 추가된 전문화(Pro) 기능

### 1. 트렌드 레이더 (`trend_radar.py`)
더 이상 감으로 상품을 고르지 않습니다.
- 구글 트렌드에서 대한민국 실시간 핫이슈 상위 10개를 자동으로 수집합니다.
- 수집된 이슈를 AI(Gemma)에게 분석시켜, **당장 쿠팡에서 잘 팔릴만한 상품 3개**를 자동으로 추천받습니다. (예: "나혼자산다 방영" -> "1인용 화로대" 추출)

### 2. 쿠팡 매처 (`coupang_matcher.py`)
- AI가 추천한 키워드를 받아, 가장 적합한 검색 결과를 연결합니다.
- (API 키가 없더라도 동작하도록, 검색 결과 URL을 자동으로 생성해 주는 Fallback 모드를 적용했습니다.)

### 3. 멀티 파이프라인 엔진 (`coupang_agent.py`)
- 이제 에이전트 하나만 실행하면, AI가 뽑아준 **3개의 아이템을 순서대로 하나씩 영상으로 제작(총 3개)**합니다.
- 제작이 완료되면 유튜브에 바로 복사/붙여넣기 할 수 있는 **'고정 댓글용 문구'**까지 자동으로 뱉어냅니다.

---

## 🛠️ 직접 실행해보기

모든 것이 연결되었습니다. 터미널(명령 프롬프트) 창에서 아래 명령어만 입력하시면, 스스로 트렌드를 찾고 3개의 영상을 순서대로 만들기 시작합니다.

```powershell
python src/luna-agent/coupang_agent.py
```

### 💡 꿀팁: 특정 방송 테마로 3개 뽑기
만약 인공지능에게 맡기지 않고, 본인이 원하는 방송이나 특정 주제로 3개를 연속으로 만들고 싶다면 아래처럼 띄어쓰기로 여러 개를 적어주시면 됩니다.

```powershell
python src/luna-agent/coupang_agent.py --product "전현무 캠핑의자" "박나래 다이어트 간식" "기안84 러닝화"
```

실행하시면 3개의 영상이 완성되며, 터미널에 유튜브 업로드용 '고정 댓글' 내용이 출력됩니다! 영상이 잘 만들어지는지 바로 확인해 보세요.
