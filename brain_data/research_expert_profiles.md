# 🌟 퀀트 및 에이전트 엔지니어링 전문가 네트워크 (Expert Profiles)

Rhumb 저장소를 중심으로 발견한, 우리가 팔로우하고 기술을 흡수해야 할 전문가 및 프로젝트 목록입니다.

## 1. 주요 개발자 및 조직
- **`supertrained`**: Rhumb의 창시자. MCP(Model Context Protocol)와 신뢰 기반 에이전트 게이트웨이 설계의 선구자.
- **`shanraisshan`**: 'Claude Code Best Practice'를 통해 에이전트의 코드 품질과 'Vibe Coding' 기법을 전파.
- **`Alien-Eyes`**: 에이전트의 판단을 독립적으로 감사하는 오픈소스 프로젝트.

## 2. 벤치마킹할 핵심 저장소 (Repositories to Watch)
- [supertrained/rhumb](https://github.com/supertrained/rhumb): 에이전트 툴 통합 프레임워크.
- [shanraisshan/claude-code-best-practice](https://github.com/shanraisshan/claude-code-best-practice): 에이전트 성능 극대화 가이드.
- [awesome-mcp-servers](https://github.com/punkpeye/awesome-mcp-servers): 에이전트 기능 확장을 위한 MCP 서버 모음.

## 3. 팔로잉 추천 전략
- 이들의 **Stargazers**와 **Followers**를 주기적으로 추적하여, 새롭게 등장하는 '에이전트 지갑(Agent Wallet)'이나 '자율 수익화 봇'의 소스코드를 수집하십시오.
- 특히 **MCP 기반의 서버**들이 출시될 때마다 우리 Alpha Agent에 즉시 연동하여 능력을 확장합니다.
