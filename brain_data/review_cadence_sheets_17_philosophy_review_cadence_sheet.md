# Review Cadence Sheet - Philosophy

## Purpose
- Define when manual review checks must occur so drafting never outruns verification.

## Cadence
- Before batch start: scope and source check
- Mid-batch: terminology and duplicate check
- End of batch: audit template + claim validation pass
- Every 1000 entries: coverage rebalance and misconception review
- At 5000 entries: final full-subject archival audit
