# Source Class Matrix - Biology

## Domain Type
- technical

## Source Matrix
| Tier | Source class | Use level |
|---|---|---|
| Primary | Standards/specifications and official technical documentation | preferred |
| Primary | Textbooks and university-level course materials | preferred |
| Secondary | Peer-reviewed papers and high-quality survey references | preferred |
| Tertiary | Strong encyclopedic references only for orientation, not sole proof | preferred |

## Restrictions
- Do not rely on a single weak source for contested claims.
- Do not treat popularity of repetition as evidence quality.
- Do not use low-trust summaries as the sole basis for entries.

## Batch Use Rule
- Record which source classes supported each archival batch.
