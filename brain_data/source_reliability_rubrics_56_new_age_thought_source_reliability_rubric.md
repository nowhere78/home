# Source Reliability Rubric - New age thought

## Purpose
- Provide a repeatable way to judge whether a source is strong enough to support archival entries in this subject.

## Reliability Dimensions
- Provenance clarity: /5
- Subject authority: /5
- Corroboration availability: /5
- Bias/agenda transparency: /5
- Technical or historical precision: /5
- Relevance to this exact subject: /5

## Rule
- Low-scoring sources may guide exploration but should not anchor high-confidence claims.
