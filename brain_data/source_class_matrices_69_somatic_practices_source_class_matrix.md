# Source Class Matrix - Somatic practices

## Domain Type
- practical

## Source Matrix
| Tier | Source class | Use level |
|---|---|---|
| Primary | Manuals, safety references, and expert-practitioner materials | preferred |
| Secondary | Historical and regional reference works | preferred |
| Secondary | Applied training materials with clear provenance | preferred |
| Tertiary | Reference summaries only after stronger validation | preferred |

## Restrictions
- Do not rely on a single weak source for contested claims.
- Do not treat popularity of repetition as evidence quality.
- Do not use low-trust summaries as the sole basis for entries.

## Batch Use Rule
- Record which source classes supported each archival batch.
