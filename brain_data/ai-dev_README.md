# AI Tooling: AI-Augmented Development (React 2026)

A professional React + React Router 7 starter focused on the **AI Engineer** role:
code generation, optimization, debugging, validation, and framework-level architecture.

## Why this project

This project translates your requirements into a practical implementation:

- **AI tooling fluency:** structured output pipeline (prompt -> validated schema -> UI).
- **Agentic development mindset:** generate plans and operational playbooks.
- **Validation and quality:** schema validation via `zod` and TypeScript strictness.
- **Modern framework adoption:** React + React Router 7 route loaders and nested layouts.
- **Production UX basics:** caching-ready data layer, modular feature structure.

## Professional React Table of Contents (2026)

1. Foundations: React architecture, strict TypeScript, and project conventions.
2. Routing Mastery: React Router 7 loaders/actions, nested layouts, data dependencies.
3. AI Integration: LLM APIs, prompt contracts, structured outputs, and schema validation.
4. Agentic Coding: automated planning, execution loops, CI hooks, and tool orchestration.
5. Reliability: caching, retries, graceful fallbacks, and observability.
6. Security & Validation: prompt safety, code review, guardrails, and evaluation suites.
7. Delivery at Scale: design systems, monorepos, release workflows, and governance.

## Quick start

```bash
npm install
npm run dev
```

Then open <http://localhost:5173>.

## Scripts

- `npm run dev` - local development
- `npm run build` - type check + production build
- `npm run typecheck` - TypeScript checks
- `npm run lint` - linting
- `npm run preview` - preview production build
