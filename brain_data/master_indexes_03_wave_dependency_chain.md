# Wave Dependency Chain

1. Wave1 strategy must exist before Wave2 module cards.
2. Wave2 cards inform Wave3 step import order and risk plans.
3. Wave3 plans feed Wave4 adapter templates and file order.
4. Wave4 templates drive Wave5 diff plans and rollout packets.
5. Wave5 execution plans are validated by Wave6 scorecards and traceability.
6. Wave7 binds all prior waves into concrete backlog execution.
