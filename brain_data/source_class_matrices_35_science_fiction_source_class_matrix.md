# Source Class Matrix - Science fiction

## Domain Type
- creative

## Source Matrix
| Tier | Source class | Use level |
|---|---|---|
| Primary | Craft manuals, production notes, practitioner interviews, and studio references | preferred |
| Secondary | Historical surveys and critical studies | preferred |
| Secondary | Museum, archive, or collection notes | preferred |
| Tertiary | Reference summaries only for navigation | preferred |

## Restrictions
- Do not rely on a single weak source for contested claims.
- Do not treat popularity of repetition as evidence quality.
- Do not use low-trust summaries as the sole basis for entries.

## Batch Use Rule
- Record which source classes supported each archival batch.
