# Batch Execution Worksheet - Cooking

## Purpose
- Drive manual archival entry creation in tightly controlled 500-entry batches.

## Batch Workflow
1. Select active batch range.
2. Review taxonomy plan, scope dossier, verification plan, source matrix, uniqueness map, and dedup ledger.
3. Draft candidate entries manually.
4. Run duplicate/similarity review.
5. Perform claim validation log updates.
6. Complete batch audit template.
7. Update subject shell and master progress tracker.
