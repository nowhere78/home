# Utilities Repository

This repository contains reusable utility functions that help streamline development. These functions cover common tasks such as data processing, formatting, and API handling.

## 📌 Purpose
- Store commonly used helper functions for easy reuse.
- Keep code DRY by centralizing utility functions.
- Improve project efficiency by maintaining a well-organized library of utilities.

## 📂 Structure
- datasource/ → Functions related to data processing and transformations with mongoDB.
- notifications/ → Functions related to notifications, such as sending emails and OTPs to phones.

## ✅ How I Use This Repo
- Continuously refine and expand utility functions as needed.
- Maintain clear documentation for each function.
- Ensure all utilities are tested before adding them to the repository.

## 🚀 Future Plans
- Add TypeScript support for better type safety.
- Create a well-structured testing suite for reliability.
- Optimize performance where applicable.
