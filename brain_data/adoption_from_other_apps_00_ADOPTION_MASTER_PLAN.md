This plan maps real reusable code from `code_of_other_apps_that_can_be_adopted/` into the Viking Girlfriend roadmap implementation.

## Target Areas
- State machines: bio, emotion, fate, metabolism integration.
- Memory: episodic storage, retrieval, summarization, context packing.
- Security: resilience, guard rails, path/input sanitization.
- Routing: local/cloud provider clients and fallback behavior.
- Observability: per-turn structured logs and failure diagnostics.

## Delivery Sequence
1. Integrate memory stack foundation.
2. Integrate bio + emotion + wyrd engines.
3. Integrate security guard and logging.
4. Integrate prompt synthesis and routing adapters.
5. Add voice and advanced architecture pieces last.
