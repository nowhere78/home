# Semantic Density Checklist - Astrology

## Purpose
- Ensure each archival entry carries meaningful information instead of shallow paraphrase or quota-padding.

## Checks
- [ ] Entry contains a real differentiating fact or distinction
- [ ] Entry is not just a reworded synonym of another item
- [ ] Entry adds domain-specific value, not generic filler
- [ ] Entry uses exact terminology where appropriate
- [ ] Entry includes caution/boundary note when needed

## Rule
- If an entry cannot justify its informational uniqueness, it should be merged, rewritten, or omitted.
