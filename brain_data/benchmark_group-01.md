# Group 1: Forms & Interactions

### 1.1 Fill a text input
Locate an input field and enter text into it.

**Verify**: Input field contains the entered text.

### 1.2 Click a button
Click a submit or action button on the page.

**Verify**: Button click triggers expected behavior (form submit or state change).

### 1.3 Use search functionality
If a search box exists, enter a query and submit.

**Verify**: Search results or filtered content appears.

### 1.4 Handle dynamic content
Wait for or interact with dynamically loaded content.

**Verify**: Dynamic content is accessible after loading.

### 1.5 Navigate back
Use browser back navigation to return to a previous page.

**Verify**: Previous page content is displayed.
