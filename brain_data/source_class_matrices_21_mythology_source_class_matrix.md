# Source Class Matrix - Mythology

## Domain Type
- spiritual

## Source Matrix
| Tier | Source class | Use level |
|---|---|---|
| Primary | Primary texts, liturgies, inscriptions, or canonical sources where applicable | preferred |
| Secondary | Scholarly religious studies and history references | preferred |
| Secondary | Tradition-specific practitioner sources with explicit labeling | preferred |
| Tertiary | Reference works only when grounded to stronger sources | preferred |

## Restrictions
- Do not rely on a single weak source for contested claims.
- Do not treat popularity of repetition as evidence quality.
- Do not use low-trust summaries as the sole basis for entries.

## Batch Use Rule
- Record which source classes supported each archival batch.
