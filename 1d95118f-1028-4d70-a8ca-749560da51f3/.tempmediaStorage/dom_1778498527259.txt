name
codex-r
description
One-time installer/repair workflow for `codex -r`: a Claude Code session picker that imports the selected Claude session into Codex while keeping `--list` and `--dry-run` safe. Use when setting up, fixing, auditing, or regenerating the Codex Claude session bridge from the existing setup log.
user-invocable
true
argument-hint
[install|repair|status]
CODEX-R
[125] (313,628)<a aria-label='Permalink: CODEX-R' class='anchor' href='#codex-r' id='user-content-codex-r' />
Install or repair
codex -r
so a user can pick an existing Claude Code session, import it into Codex, and start the imported Codex session immediately.
This is a Markdown-only skill. Do not treat the skill folder as an installer package. Reconstruct or patch the local files directly from the source log and the current machine state.
If a local setup log exists at
~/ws/codex-claude-session-import-setup-log.md
, read it first. If it does not exist, follow this skill as the source of truth.
