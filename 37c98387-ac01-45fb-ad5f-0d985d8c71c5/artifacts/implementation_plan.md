# Instagram/TikTok 자동 육성 봇 (AppAgent 기반) 구축 계획

사용자님의 V60 스마트폰을 활용하여 아침 9시까지 인스타그램/틱톡 채널을 자동으로 육성하는 AI 봇을 구축합니다.

## ⚠️ User Review Required (중요 확인 사항)
> [!IMPORTANT]
> **지인 연락처/계정 상호작용 완벽 차단 방안**
> 지인과의 상호작용을 막기 위해 봇은 **"홈(Home) 피드"를 절대 보지 않습니다.**
> 대신 봇은 무조건 **"검색(돋보기) 탭"**으로 이동하여 특정 해시태그(예: `#홈카페`, `#IT기기`, `#맛집` 등)를 검색한 뒤, 모르는 사람들의 게시물에만 들어가서 좋아요와 댓글을 남기도록 코드를 설계할 예정입니다. 
> 
> **질문:** 봇이 주로 검색하고 활동했으면 하는 타겟 해시태그(관심사)가 있나요? (예: 자동차, 요리, 전자기기 등) 없으시면 제가 임의로 무난한 일상 해시태그로 설정하겠습니다.

## Proposed Changes

### 1. 무한 루프 자동화 스크립트 제작
현재 AppAgent의 기본 데모 스크립트는 1회성 명령만 수행하고 종료됩니다. 이를 아침 9시까지 쉬지 않고 작동하게 만들려면 새로운 파이썬 래퍼(Wrapper) 스크립트가 필요합니다.

#### [NEW] `C:\Users\smile\AppAgent\auto_social_bot.py`
- 현재 시간부터 오전 9시까지 `while` 루프를 도는 파이썬 스크립트.
- AI에게 다음과 같은 프롬프트를 주기적으로 주입합니다:
  - "인스타그램을 열어라."
  - "돋보기 모양(검색)을 눌러 특정 해시태그를 검색해라."
  - "게시물을 클릭하고, 사진을 분석해서 자연스러운 칭찬 댓글을 달아라."
  - "뒤로 가기를 누르고 다음 게시물로 스크롤해라."

### 2. ADB 무선 연결 상태 점검
- 봇을 실행하기 전, V60의 무선 ADB 연결(`192.168.123.108:5555`)이 안정적으로 유지되고 있는지 한 번 더 테스트합니다.

## Verification Plan

### Manual Verification
1. 스크립트 실행 직후, 사용자님께서 V60 화면을 직접 보며 AI가 인스타그램 홈 피드가 아닌 **검색창**으로 제대로 향하는지 확인합니다.
2. AI가 모르는 사람의 글에 자연스럽게 댓글을 달고 다음 글로 넘어가는 패턴이 2~3회 정상 작동하는지 모니터링합니다.
3. 이상이 없다면 아침 9시까지 스마트폰을 그대로 방치하시면 됩니다!
