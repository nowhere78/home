# 🎯 일일 포스팅 일괄 E-E-A-T 보완 및 예약 발행 마스터플랜 (/goal)

회원님의 지시(`/goal`)에 따라 76번 포스팅 복구를 진행함과 동시에, **"오늘 올라간 모든 글들을 찾아내어 고퀄리티 이미지와 E-E-A-T 구조로 뜯어고친 뒤, 76번 글과 함께 예약 발행으로 돌리는"** 대규모 자동화 파이프라인 구축 계획을 세웠습니다.

이 작업은 기존 단일 스크립트들을 하나로 엮는 거대한 오케스트레이션(Orchestration) 작업이므로 아래 계획을 검토 후 승인해주시기 바랍니다.

## 🚨 해결해야 할 기술적 과제
1. **오늘 올린 글 ID 추출:** 파이썬 셀레니움으로 `story73341.tistory.com/manage/posts` 에 접속하여 오늘 날짜로 올라간 글들의 고유 ID 번호 리스트를 긁어오는 새 스크립트가 필요합니다.
2. **기존 글 내용 추출 및 E-E-A-T 변환:** 해당 ID의 수정 페이지로 들어가 기존 원고를 긁어온 뒤, 로컬 Llama 3.1 모델을 돌려 "어머니 경험담, 쿠팡 링크, 학회 출처"가 포함된 완벽한 템플릿으로 다시 써서(Rewriting) 저장해야 합니다.
3. **Kaggle 2.0 이미지 자동 삽입:** 재생성된 원고를 바탕으로 SDXL 프롬프트를 뽑아내고 썸네일을 생성한 뒤, 파이썬 클립보드 로직으로 본문에 삽입해야 합니다.
4. **예약 발행(Scheduling) 로직 개발:** 기존 `tistory_updater.py`는 오직 '발행(공개/비공개)' 버튼만 눌렀습니다. 이를 개조하여 **'예약' 라디오 버튼을 클릭하고 원하는 날짜와 시간을 세팅**하는 고난도 DOM 제어 로직을 추가해야 합니다.
