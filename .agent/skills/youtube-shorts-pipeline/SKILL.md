# 🎬 유튜브 쇼츠 원클릭 자동 생성 스킬 지침서

이 스킬은 `src/luna-agent/luna_shorts_pipeline.py` 스크립트를 실행하여 인터넷 기사를 수집하고, 대본을 작성하고, Edge TTS로 음성을 입히고, 배경 영상을 조립하여 최종 `.mp4` 파일을 만들어내는 역할을 수행합니다.

## 🎯 실행 조건
사용자가 "오늘자 쇼츠 만들어줘", "[주제]로 쇼츠 하나 뽑아줘" 라고 요청할 때 이 스킬을 발동합니다.

## 📋 실행 절차 (Step-by-Step)

### 1단계: 사용자 의도 파악
- 사용자가 특정 주제(예: "일론 머스크 화성 이주")를 주었는지 확인합니다.
- 주제가 없다면 파이프라인이 알아서 최신 AI/테크 뉴스를 가져오도록 인자 없이 실행을 준비합니다.

### 2단계: 파이프라인 스크립트 실행
- 터미널을 열고 다음 명령어를 통해 `luna_shorts_pipeline.py`를 실행합니다.
  - 주제가 없을 때: `python src\luna-agent\luna_shorts_pipeline.py`
  - 주제가 있을 때: `python src\luna-agent\luna_shorts_pipeline.py 일론 머스크 화성 이주`
- 실행 시 백그라운드로 보내지 말고(`WaitMsBeforeAsync`를 적절히 주거나), 결과가 나올 때까지 기다리면서 진행 상황을 관찰합니다. (일반적으로 30~60초 소요)

### 3단계: 결과물 검증
- 파이프라인이 완료되면 `output/shorts/` 폴더 내에 가장 최근에 생성된 타임스탬프 폴더를 찾습니다.
- 해당 폴더 안에 `.mp4` 영상 파일과 `voiceover.mp3` 오디오 파일이 정상적으로 생성되었는지 확인합니다. (파일 사이즈 체크)

### 4단계: 보고 및 전달
사용자에게 다음과 같이 보고합니다.

```markdown
# 🎬 쇼츠 영상 제작 완료!

- **주제**: (스크립트 실행 결과로 나온 주제)
- **저장 위치**: `output/shorts/[날짜폴더]/[파일명].mp4`
- **검증**: 오디오(Edge TTS)와 자막 병합이 정상적으로 처리되었습니다.

영상을 확인해 보시고, 마음에 드신다면 유튜브 채널에 업로드해 주세요!
```
