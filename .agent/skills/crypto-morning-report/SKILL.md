# 🌅 아침 시황 분석 보고서 (Crypto Morning Report) 스킬 지침서

이 스킬은 사용자가 퀀트 트레이딩 봇(Luna v5.1)의 상태와 주요 암호화폐(BTC, ETH, XRP 등)의 시황을 매일 아침 빠르고 직관적으로 파악할 수 있도록 돕는 자동화 리포트 생성 스킬입니다.

## 🎯 실행 조건
사용자가 "아침 코인 시황 알려줘", "오늘의 보고서 작성해줘", "현재 상태 브리핑해줘" 라고 요청할 때 이 스킬을 발동합니다.

## 📋 실행 절차 (Step-by-Step)

### 1단계: 실시간 시장 데이터 수집
- 파이썬 코드를 백그라운드에서 실행하여 업비트(Upbit) API 또는 `pyupbit` 라이브러리를 통해 주요 자산(BTC, ETH, XRP)의 현재가, 24시간 변동률, 거래량 데이터를 가져오세요.
- (선택) `Fear & Greed Index` (공포 탐욕 지수) 데이터를 웹 검색으로 확인하세요.

### 2단계: Trading Bot (Luna v5.1) 상태 확인
- 사용자의 포트폴리오 잔고(`pyupbit.get_balances()` 등 활용)를 조회하여 현재 현금(KRW) 비중과 코인 투자 비중을 계산하세요.
- `docs/intelligence/quant_research/trading_log_v5.txt` 파일의 마지막 10줄을 읽어 밤사이 봇이 매수/매도를 진행했는지 요약하세요.

### 3단계: 보고서 작성 및 출력
아래의 양식에 맞추어 Markdown 형식으로 사용자에게 직접 브리핑합니다.

```markdown
# 📊 [오늘 날짜] Luna v5.1 모닝 브리핑

## 1. 밤사이 시장 동향 (Market Overview)
- **BTC**: 현재가 / 24h 변동률
- **ETH**: 현재가 / 24h 변동률
- **XRP**: 현재가 / 24h 변동률
- **시장 분위기**: 공포/탐욕 지수 또는 전반적인 추세 요약

## 2. 퀀트 봇(v5.1) 작동 로그 요약
- 어젯밤 주요 매매 내역: (예: 02:30 BTC 50만원 매수 승인)
- 현재 포트폴리오 비율: (현금 40%, 코인 60%)

## 3. Luna의 오늘 투자 전략 제언
- (데이터를 바탕으로 오늘 공격적으로 갈지, 보수적으로 현금을 보유할지 한 줄 코멘트)
```

### 4단계: 마무리
보고서를 출력한 후, "봇을 강제로 리밸런싱 할까요?" 혹은 "다른 뉴스 검색이 필요하신가요?" 등의 후속 질문을 남기며 대기를 종료합니다.
