# 🏛️ Genspark 기반 초고화질 리얼리즘 시나리오 & 제작 기술 (TITANIC Special)

본 문서는 Genspark 에이전트가 제안한 '타이타닉 생존 가이드' 스타일의 고전환율 시나리오 구성 및 영상 기술을 정리한 것입니다.

## 1. 고속 페이싱 시나리오 구조 (총 85초)
*   **0-5s (Hook)**: 극적인 수치(67% -> 94%)와 타임 리미트(2분 45초)를 제시하여 즉각적인 몰입 유도.
*   **5-25s (Point 1)**: 대중적 상식을 뒤집는 반전(얼음산 vs 과속) 제시 + 수치 데이터(23노트) 활용.
*   **25-50s (Point 2)**: 구체적인 행동 지침(왼쪽 갑판 vs 오른쪽 갑판)과 시간적 손실 데이터 제공.
*   **50-70s (Point 3)**: 심리적 반전(평온함 = 위험)과 역사적 교훈(무시된 SOS) 연결.
*   **70-85s (Conclusion)**: 핵심 내용 요약 + 호기심 유발형 다음 예고(부자 vs 빈자) + 강한 CTA.

## 2. 화면 연출 및 편집 기술 (Segment Mapping)
| 시간 | 화면 전환 | 효과 및 자막 스타일 |
| :--- | :--- | :--- |
| 0-3s | 바다 줌인 | 짙은 블루 필터 + **"⚠️ 당신의 생존 확률"** (대왕 자막) |
| 3-8s | 속력계 애니메이션 | 바늘 움직임 + **빨간색 강조 자막** |
| 8-15s | 좌우 분할 화면 | 초록/주황 대비 + **"생명의 분기점"** |
| 15-50s | 빠른 프레임 전환 | 0.5초 간격 컷 + **깜박이는 경고 자막** |
| 50-85s | 시뮬레이션/통계 | 막대 그래프, 붉은 화살표 등 **데이터 시각화** |

## 3. 썸네일 전략 (3-Tier Design)
*   **Option 1 (Shock)**: 극단적 결과 수치 강조 ("생존율 94%").
*   **Option 2 (Data)**: 변화 과정 시각화 ("67% → 94%").
*   **Option 3 (Question)**: 시청자 개입 유도 ("당신이라면?").

## 4. 로컬 AI 적용 기술 (Segment-based Rendering)
*   **분할 렌더링**: 85초 영상을 한 번에 뽑지 않고, 8초 내외의 고품질 세그먼트 8개로 나누어 생성 후 병합.
*   **BGM 변곡점**: 후반부로 갈수록 퍼커션과 오르간 사운드를 추가하여 긴장감을 점진적으로 고조.
*   **데이터 오버레이**: 단순 이미지가 아닌 통계 그래프나 시뮬레이션 데이터를 이미지 위에 합성.
