## Findings
### 1. Generative Agents (AI Village)
- Inspired by Stanford/Google research.
- 25 AI residents with unique personalities, memories, and relationships.
- They communicate and interact in a physical environment autonomously.

### 2. The AI Brain Architecture
- **Perceive**: Agents sense their environment and record it.
- **Memory Stream**: A long-term database storing all observations as natural language.
- **Retrieval Engine**: Filters memories using a "Retrieval Score" based on:
    - **Recency**: How recent the memory is.
    - **Importance**: How significant the event was (scored 1-10).
    - **Relevance**: How related it is to the current situation.
- **Reflection (성찰)**: 
    - Periodic synthesizing of memories into higher-level insights.
    - Uses "Tree of Thoughts" (Data -> Intermediate thoughts -> Core beliefs).
- **Planning (계획)**:
    - Top-down approach: Daily schedule -> Hourly -> Minute-by-minute.
    - Dynamic adjustment based on real-time observations.

### 3. Vision
- Moving beyond "NPCs" to "AI Producers".
- Simulating human life for social system design, personalized assistants, and immersive virtual realities.
- The goal is for AI to understand memory and reflection like a "mirror" to humans.
