## 📈 3. 업로드 및 최적화 전략

쇼츠 조회수와 클릭률(CTR)을 극대화하기 위한 단계별 방법입니다:

### 1단계: 채널 만들기
1. 유튜브 접속 후 프로필 아이콘 클릭 ➡️ **채널 만들기** 선택
2. 프로필 사진으로 과자 아이콘 이미지를 사용 (예: 쿠키 이모지 또는 사이트 로고 크롭 이미지)

### 2단계: 영상 업로드
1. 유튜브 모바일 앱 또는 크리에이터 스튜디오에서 `shorts.mp4` 업로드
2. **제목**: 클릭을 유도하는 제목과 인기 해시태그 사용
   - *예시*: `오늘 먹을 과자 골라주는 룰렛 🍿 #shorts #과자추천 #결정장애`
3. **배경음악**: 모바일 앱에서 업로드 시 **인기 트렌드 음악을 추가**하세요. 트렌드 오디오는 알고리즘 도달률을 크게 높여줍니다.

### 3단계: 댓글에 링크 고정 (핵심!)
유튜브는 더 이상 쇼츠 설명란이나 제목에 클릭 가능한 링크를 허용하지 않습니다. **댓글에 링크를 남기고 고정**하는 것이 유일한 방법입니다:
1. 영상에 댓글 작성:
   - `오늘 먹을 과자 룰렛으로 정하러 가기! 👉 https://nowhere78.github.io/snack-picker/`
2. 해당 댓글을 **댓글 고정**하면 항상 최상단에 표시됩니다.

## 🛠️ 4. 나중에 새 영상 만들기

새로운 영상이 필요할 때(다른 룰렛 결과나 다른 제목으로) 아래 명령어를 실행하면 됩니다:
```bash
node record_shorts.js
```
브라우저를 자동으로 실행하고, 룰렛을 돌리고, 9.5초짜리 프레임을 녹화한 뒤 새로운 `shorts.mp4`를 생성합니다!
