# 🚀 주식 자동 매매 전략 및 API 비교 분석

이 문서는 주식 시장 자동화를 위한 최적의 전략과 실행 도구(API)를 비교 분석하여 제안합니다.

## 1. 추천 자동 매매 전략 (Top 3)

### ① 퀀트 모멘텀 전략 (Quant Momentum)
*   **원리**: 최근 3~6개월간 상승세가 강했던 종목들을 필터링하여 매수.
*   **장점**: 추세가 확실한 종목에 올라타므로 수익 극대화 가능.
*   **적합**: 미국 나스닥 종목 및 국내 코스피 대형주.

### ② 가치 투자 기반 스크리닝 (Value-Based Screening)
*   **원리**: PER, PBR, ROE 등 재무 지표가 저평가된 종목을 자동으로 수집하여 포트폴리오 구성.
*   **장점**: 하락장에서의 방어력이 좋으며 장기적인 우상향 기대.
*   **적합**: 국내 배당주 및 가치주.

### ③ 변동성 돌파 전략 (Volatility Breakout)
*   **원리**: 전일 가격 변동폭의 일정 비율 이상을 돌파할 때 매수하여 당일 매도.
*   **장점**: 자금 회전율이 높고 리스크 관리가 쉬움 (업비트 봇에 적용된 것과 유사).
*   **적합**: 단기 변동성이 큰 테마주 및 급등주.

## 2. 주식 자동 매매 API 비교

| API 구분 | 장점 | 단점 | 추천 용도 |
| :--- | :--- | :--- | :--- |
| **한국투자증권 (KIS API)** | REST API 지원으로 파이썬 연동이 매우 쉬움. 국내/미국 주식 모두 지원. | 토큰 갱신 로직 필요. | **입문자 및 파이썬 개발자 최우선 추천** |
| **키움증권 (Open API+)** | 가장 오래된 서비스로 커뮤니티 예제가 방대함. 안정성 높음. | Windows 32bit 환경 필수, COM 방식이라 파이썬 연동이 복잡함. | 대량 주문 및 복잡한 HTS 연계 작업 |
| **Alpaca (Global)** | 미국 주식 전문. 수수료 무료 모델 제공. API 디자인이 매우 현대적임. | 국내 주식 미지원. | **미국 주식 전용 자동 매매** |
| **Yahoo Finance (yf)** | 별도 가입 없이 시세 데이터 수집 가능 (무료). | 실시간 주문 불가, 데이터 지연(15분) 및 크롤링 차단 위험. | **백테스트 및 기초 데이터 수집용** |

## 3. 실행 로드맵 제안

1.  **Phase 1 (완료)**: 기초 데이터 수집 파이프라인 구축 (`stock_data_collector.py`).
2.  **Phase 2 (완료)**: 개별 종목 재무 및 기술적 분석 리포트 자동화 (`stock_analyst.py`).
3.  **Phase 3 (다음 단계)**: **한국투자증권 API** 계좌 연동 및 모의투자 환경 설정.
4.  **Phase 4**: 퀀트 전략을 코드로 구현하여 실제 매수/매도 시뮬레이션.
