# 구글 클라우드 콘솔 및 YouTube API 설정 가이드

## **1단계 : 구글 클라우드 콘솔 접속 및 프로젝트 생성**

AI 에이전트가 활동할 수 있는 가상의 작업실(프로젝트)을 구글에 만들어 주는 단계입니다.

1. **구글 클라우드 콘솔 접속하기**
    - 링크: [Google Cloud Console (클릭)](https://console.cloud.google.com/)
    - 본인의 구글 계정으로 로그인합니다. (유튜브 채널이 있는 계정을 추천합니다.)
2. **새 프로젝트 만들기**
    - 화면 왼쪽 상단의 **[프로젝트 선택]** (또는 기존 프로젝트 이름)을 클릭합니다.
    - 우측 상단의 **[새 프로젝트]** 버튼을 누릅니다.
    - **프로젝트 이름:** 본인이 알아보기 쉬운 이름으로 입력합니다. (예: `YouTube-Auto-Agent-Luna`)
    - **[만들기]** 버튼을 클릭합니다.
    - *💡 잠시 기다리면 프로젝트가 생성됩니다. 상단 알림창에서 방금 만든 프로젝트를 선택(Select Project)해 주세요.*

---

## **🔌 2단계 : YouTube Data API v3 활성화 (Enable API)**

에이전트가 유튜브 시스템(업로드, 댓글 등)을 합법적으로 제어할 수 있도록 스위치를 켜주는(Enable) 단계입니다.

1. **API 라이브러리 이동**
    - 클라우드 콘솔 왼쪽 메뉴 햄버거 버튼(☰) 클릭 👉 **[API 및 서비스]** 👉 **[라이브러리]**를 클릭합니다.
2. **유튜브 API 검색 및 활성화**
    - 검색창에 `YouTube Data API v3`를 검색합니다.
    - 검색 결과에 나온 **[YouTube Data API v3]**를 클릭합니다.
    - 파란색 **[사용(Enable)]** 버튼을 클릭합니다.
    - *💡 이제 여러분의 프로젝트가 유튜브 시스템과 소통할 준비를 마쳤습니다!*

---

## **🔐 3단계 : OAuth 동의 화면 구성 (Consent Screen)**

에이전트가 내 유튜브 채널에 접근할 때 "이거 내가 허락한 시스템 맞아!"라고 구글에 알려주는 서약서 작성 단계입니다.

1. **동의 화면 설정 진입**
    - 왼쪽 메뉴 👉 **[API 및 서비스]** 👉 **[OAuth 동의 화면]**을 클릭합니다.
2. **사용자 유형 선택**
    - **[외부(External)]**를 선택하고 **[만들기]**를 누릅니다. (본인 계정으로만 테스트할 것이므로 외부에 노출되지 않습니다. 안심하세요!)
3. **앱 정보 입력 (필수 항목만 작성)**
    - **앱 이름:** `Luna Agent` (또는 원하는 이름)
    - **사용자 지원 이메일:** 본인 구글 이메일 선택
    - 화면 맨 밑의 **개발자 연락처 정보:** 본인 이메일 입력
    - **[저장 후 계속]**을 클릭합니다.
4. **범위(Scopes) & 테스트 사용자(Test Users)**
    - 범위 단계는 그냥 **[저장 후 계속]**을 누릅니다.
    - 테스트 사용자 단계에서 **[+ ADD USERS]** 버튼을 누르고, **자신의 구글 이메일 주소(유튜브 채널 소유자 계정)**를 입력 후 **[추가]**, 그리고 **[저장 후 계속]**을 누릅니다.

---

## **🔑 4단계 : 마스터 키(자격 증명/Credential) 발급받기**

가장 중요한 단계입니다! 에이전트의 안티그래비티 파일(파이썬 코드 등)에 심어줄 진짜 비밀번호를 발급받습니다.

1. **자격 증명 메뉴 진입**
    - 왼쪽 메뉴 👉 **[API 및 서비스]** 👉 **[자격 증명(Credentials)]** 메뉴로 이동합니다.
2. **OAuth 2.0 클라이언트 ID 만들기**
    - 상단의 **[+ 자격 증명 만들기]** 클릭 👉 **[OAuth 클라이언트 ID]** 선택.
3. **애플리케이션 유형 선택**
    - **[데스크톱 앱 (Desktop App)]** 을 선택합니다. (서버/파이썬 등에서 돌릴 때 가장 직관적입니다.)
    - 이름은 알아보기 쉽게 입력하고 **[만들기]**를 누릅니다.
4. **💡 [매우 중요] 보안 JSON 파일 다운로드**
    - 화면에 클라이언트 ID와 보안 비밀이 팝업으로 뜹니다.
    - 창 하단이나 목록 우측에 있는 **[⬇️ JSON 다운로드]** 아이콘을 클릭하여 파일을 PC에 저장합니다.
    - 파일 이름을 `client_secrets.json` 과 같이 깔끔하게 변경한 뒤, 에이전트가 실행되는 폴더(작업 공간) 안에 넣어두시면 됩니다!

다운로드하신 `json` 파일이나 API 키 값은 여러분 채널 현관문의 마스터키입니다.

- **절대 깃허브(GitHub) 등 공개된 웹사이트나 커뮤니티에 이 파일을 포함하여 코드를 업로드하지 마세요.**
- 타인에게 공유하지 마시고 오직 로컬 폴더(환경변수 등) 안에서만 안전하게 보관하세요.
