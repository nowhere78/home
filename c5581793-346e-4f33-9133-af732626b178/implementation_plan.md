# 완전 자동화 블로그 포스팅 파이프라인 구축

유튜브 영상 URL만 입력하면 자막을 추출하고, Ollama 로컬 LLM이 블로그 원고를 작성한 뒤, 네이버 블로그와 티스토리에 자동으로 업로드하는 "원클릭 완전 자동화(End-to-End)" 시스템을 구축합니다.

## User Review Required

> [!IMPORTANT]
> 로컬 환경에서 구동되는 완전 자동화 스크립트이므로, 아래 **Open Questions**에 대한 사용자님의 피드백과 결정을 바탕으로 최종 코드를 작성할 예정입니다. 

## Open Questions

> [!WARNING]
> 파이프라인 설계를 위해 다음 몇 가지 사항을 결정해 주셔야 합니다.

1. **Ollama 모델 선택:** 사용자님의 PC에 여러 모델(`qwen2.5:14b`, `gemma:7b`, `luna-expert-v5:latest` 등)이 설치되어 있는 것을 확인했습니다. 어떤 모델을 메인 글쓰기 작가로 지정할까요? (한국어 작문 능력이 뛰어난 모델을 추천합니다.)
2. **업로드 방식 결정:** 
   * A안: Ollama가 글을 다 쓰면 묻지도 따지지도 않고 바로 네이버/티스토리에 업로드(임시저장)해 버리기
   * B안: Ollama가 글을 쓰면 일단 화면에 보여주고, "업로드 할까요? (y/n)" 라고 물어본 뒤 승인하면 업로드하기
3. **블로그 동시 업로드:** 원고 하나를 네이버 블로그와 티스토리에 **동시에 둘 다** 올릴까요? 아니면 봇 실행 시 어디에 올릴지 선택하게 할까요? (완전히 똑같은 글이 두 플랫폼에 올라가면 검색 포털에서 '유사 문서(중복 글)'로 인식해 품질이 떨어질 수 있습니다.)

## Proposed Changes

새로운 워크플로우를 관장하는 메인 프로그램(`main_bot.py`)을 만들고, 각 모듈을 통합합니다.

### 1. `youtube_parser.py` 생성
#### [NEW] `youtube_parser.py`
* `youtube-transcript-api` 라이브러리를 사용하여 사용자가 입력한 유튜브 URL에서 한글 자막 스크립트를 추출하여 텍스트로 반환합니다.

### 2. `ollama_writer.py` 생성
#### [NEW] `ollama_writer.py`
* 파이썬의 `requests` 라이브러리를 사용해 사용자 PC의 로컬 Ollama API(`http://localhost:11434/api/generate`)로 유튜브 자막을 전송합니다.
* 강력한 프롬프트를 통해 "건강/정보성 블로그 포스팅 형식"으로 제목과 본문을 작성하도록 명령합니다.
* 작성된 원고를 `contents/` 하위 폴더에 타임스탬프 이름으로 저장합니다.

### 3. 기존 업로더 모듈화 수정
#### [MODIFY] `naver_uploader.py`
#### [MODIFY] `tistory_uploader.py`
* 기존 코드를 메인 스크립트에서 쉽게 호출할 수 있도록 함수화(`def auto_post_naver(folder_path):`)하고, 특정 폴더의 원고를 지정하여 업로드할 수 있도록 인자(argument)를 받게 수정합니다.

### 4. 통합 실행 스크립트 `main_bot.py`
#### [NEW] `main_bot.py`
* 전체 프로세스를 지휘하는 사령탑입니다.
* 흐름: 사용자에게 유튜브 URL 입력받음 -> 자막 추출 -> Ollama 원고 작성 대기 -> 파일 저장 -> 네이버/티스토리 업로더 자동 호출 -> 종료
