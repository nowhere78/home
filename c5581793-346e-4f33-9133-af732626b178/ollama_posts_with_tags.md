가장 아래쪽을 보시면 스크립트 수정 시 요청했던 대로 **`태그: #키워드`** 형식으로 태그가 완벽하게 추출되어 달려있는 것을 보실 수 있습니다.

## 📝 포스팅 1 (네이버 블로그용)
제목: 🪄 매번 똑같은 글이 지겹다면? 클로드 코드 글쓰기 자동화, 4단 구조로 끝내버리는 실전 가이드

여러분, 안녕하세요! 😊
오늘은 정말 흥미진진한 주제로 찾아왔어요. 매일 비슷한 톤으로 블로그 글을 쓰다 보면 "아, 또 이 얘기야?" 싶으시죠? 매번 프롬프트를 복사 붙여넣기 하기도 지치고요. 그래서 오늘은 **클로드 코드에서 글쓰기 자동화 시스템**을 어떻게 구축하는지, 영상 내용을 토대로 깔끔하게 정리해 드릴게요.

### 🤖 왜 굳이 클로드 코드로 자동화를 할까?
채팅창에서 글 쓸 때마다 톤을 설명하고, 금칙어 알려주고, 구조 잡아달라고 하기… 솔직히 번거롭잖아요. 🤯
클로드 코드를 활용하면 **내 컴퓨터 안에서 스스로 돌아가는 글쓰기 시스템**을 만들 수 있어요. 바이브 코딩만이 아니라, 워크플로우 자동화 도구로도 쓸 수 있다는 게 핵심 포인트!

### 🧱 4가지 핵심 축 — 이게 다 알아야 할 기본 구조
**1️⃣ Context (컨텍스트) — 내 글의 DNA 만들기 📝**
- 톤, 금칙어, 문체, 구조를 **마크다운 파일**로 미리 박아두는 거예요.
- 모든 세션에서 자동으로 로딩되니까 **매번 설명할 필요가 없죠**.

**2️⃣ Sub-agents (서브 에이전트) — 일 잘하는 직원 4명 고용 👥**
- **Researcher**: 자료 수집 🔍 (Haiku)
- **Planner**: 글 구조 설계 📐 (Opus)
- **Writer**: 초안 작성 ✍️ (Sonnet)
- **Reviewer**: 검수 및 다듬기 🧐 (Opus 4.7)

**3️⃣ Skills (스킬) — 한 줄로 끝내는 마법 주문 ✨**
"블로그 파이프라인" 딱 한 줄이면, 앞에서 만든 서브 에이전트들이 자동으로 순차 실행돼요.

**4️⃣ Hooks (훅) — 강제 실행 룰 ⚙️**
특정 규칙을 **모든 세션에서 강제로 적용**하게 만들어요. 

### 🚨 절대 빼먹으면 안 되는 것: 팩트 체커
영상에서 가장 강조한 부분! **할루시네이션(Hallucination)**을 막기 위해 **Fact Checker 에이전트**를 별도로 추가하는 걸 강력 추천하고 있었어요. 검증된 자료만 사용해야 신뢰도가 올라갑니다.

### 🎯 마무리하며
> **"컨텍스트 + 서브 에이전트 + 스킬 + 훅 + 팩트체커 = 나만의 글쓰기 공장"**

처음부터 완벽하게 만들려 하지 마시고, 부족한 부분을 단계적으로 보강해 나가는 게 핵심이에요!

## 📝 포스팅 2 (티스토리용)
제목: Claude Code 글쓰기 자동화, 직접 구축해본 결과를 공유합니다 (실전 워크플로우 완전 해부)

여러분, 혹시 AI한테 글 써달라고 매번 똑같은 톤 설명해주다 지치신 적 없으신가요? 😅 오늘은 제가 직접 Claude Code로 만든 글쓰기 자동화 시스템이 어떤 구조로 돌아가는지, 실제 구축 과정과 함께 솔직하게 풀어보려고 합니다.

### 🚀 왜 Claude Code로 글쓰기 자동화를 시도했을까?
채팅창에서 매번 "이 톤으로 써줘" 반복하다 보면 블로그 글 한 편 쓰는 데 톤 설명하는 시간이 절반이 차지합니다. 그래서 **"내 글쓰기 방식을 파일로 박아두면 어떨까?"** 하는 아이디어에서 출발했습니다.

### 🏗️ 시스템의 4가지 축
**1️⃣ Context (컨텍스트) — 글의 DNA 만들기**
`tone.md`, `readability.md` 등 마크다운 파일에 리듬감, 번역체 금지 등을 미리 명시해둡니다.

**2️⃣ Sub-agents (서브 에이전트) — 역할 분담 시스템**
자료 수집은 Haiku, 검수는 Opus 등 **토큰 비용 최적화**를 위해 에이전트를 나눕니다.

**3️⃣ Skill (스킬) — 원라인 실행의 마법**
"블로그 파이프라인" 이 한 줄이면 리서치부터 발행까지 쭉 돌아갑니다.

**4️⃣ Hook (훅) — 강제 규칙 (선택사항)**
"리서치할 때 반드시 최신 날짜 기준"처럼 AI가 가끔 빼먹는 부분을 강제할 수 있어요.

### ⚠️ 반드시 넣어야 할 것: 팩트 체커
**할루시네이션 체크 없으면 글 자체가 신뢰를 잃어요.** 특히 정보성 글에서는 팩트체커를 별도 에이전트로 추가하는 걸 무조건 추천드립니다.

### 📝 Notion MCP 연동으로 발행 자동화까지
네이버 블로그는 API 제약이 있고 계정 삭제 리스크도 있어서, **Notion**에 자동 저장되도록 세팅했습니다. Claude.ai에서 커넥터로 Notion을 추가하면 쉽게 연동됩니다.

### 🎯 마무리하며
완전 자동화는 아직 어렵습니다. 글이 산만해지거든요. 하지만 **시스템화해두면 톤이 흔들리지 않고**, 일관된 품질의 글을 생산할 수 있습니다.
